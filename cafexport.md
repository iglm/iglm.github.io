Aplicación profesional para la gestión de compras y comercialización de café. Lleva el control completo de tus operaciones con cálculo de **margen P&L (Profit & Loss)** en tiempo real. Diseñada para comercializadores, cooperativas y acopios.

## 🎯 Funcionalidades clave

- ✅ Registro de compras por cliente y tipo de café
- ✅ Cálculo automático de margen P&L: (Precio Nacional - Precio Compra) × Kg Pergamino Seco
- ✅ Estadísticas globales y por cliente
- ✅ Dashboard con indicadores clave de rendimiento
- ✅ Exportación a Excel y PDF
- ✅ Dark mode, Material Design 3
- ✅ Backup y restauración de base de datos
- ✅ 100% offline — sin conexión a internet

## 👥 ¿Para quién es?

- **Comercializadores y exportadores** de café
- **Cooperativas y asociaciones** de caficultores
- **Acopios rurales** que manejan compras diarias
- **Administradores** que necesitan control de margen en tiempo real

## 🛠️ Stack técnico

| Componente | Tecnología |
|---|---|
| Lenguaje | Kotlin |
| UI | Jetpack Compose + Material Design 3 |
| Arquitectura | Clean Architecture + MVVM |
| Base de datos | Room (SQLite) |
| Inyección | Hilt |
| Export | Apache POI (Excel), iText (PDF) |
