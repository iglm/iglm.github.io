# 🌱 IGLM — Software y Asesoría Agrícola

**IGLM** es un equipo de la **Universidad de Caldas** que desarrolla apps móviles profesionales para transformar la gestión del campo colombiano.

## 📱 Apps

| App | Descripción | Tecnología |
|---|---|---|
| 🌿 **Monitoreo Broca y Roya** | Evaluación árbol por árbol de broca y roya en café | React + TS + Capacitor |
| 📋 **Programador de Labores** | Planificación agrícola multi-cultivo | React + TS + Capacitor |
| ☕ **CompraCafe** | Gestión de compras de café con margen P&L | Android nativo (Jetpack Compose) |
